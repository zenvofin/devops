# Zenvofin's devops

## Requirements

- Docker
- Docker Compose
- Docker Desktop

## Installation process

1. Clone this repository to your local machine.
2. Run the Docker Compose stack: `docker compose up -d` (or `docker compose up -d --build` to rebuild images).
3. Navigate to Seq (http://localhost:5341).
